# Жүсіпбек Аймауытов
Ақпарат
Жобаның мақсаты — қазақтың біртуар тұлғасы Жүсіпбек Аймауытовқа арналған веб-сайт әзірлеу және оны интернетке жариялау.

Сайттың құрылымы HTML семантикалық тегтерін қолдана отырып жасалған және CSS арқылы безендірілген. Мазмұны тұлғаның өмірбаянын, романдары мен пьесаларын, аудармаларын және оның қоғамға қосқан үлесін сипаттайтын бес негізгі бөлімнен тұрады. Сайтқа мобильді меню және жоғарыға көтерілу батырмасы сияқты интерактивті функциялар JavaScript көмегімен қосылды.

Дайын жоба GitHub репозиторийіне жүктеліп, InfinityFree хостингі арқылы интернетке жарияланды. Нәтижесінде жұмыс істеп тұрған сайт сілтемесі мен оның бастапқы коды тапсырылады. Бұл жұмыс веб-технологияларды қолдану, нұсқаларды басқару жүйесімен жұмыс істеу және сайтты хостингке орналастыру дағдыларын толық көрсетеді.
